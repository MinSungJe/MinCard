## ⛳ 구현할 기능 목록

### 구현

- 카드 컴포넌트 구현
- **React**와 **Vite**를 이용해 카드 컴포넌트 표시

### StoryBook 라이브러리 활용

- 의도에 따라 잘 표시가 되고 있는지 **StoryBook**을 활용해 점검
  - UI 단계에서 빠른 피드백이 가능

### MSW 라이브러리 활용

- 카드 속 데이터를 **Vitest**로 구현한 Mocking API로 전달받고 이를 표시
